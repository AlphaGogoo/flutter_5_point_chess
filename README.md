# 双人联机五子棋游戏

## 功能描述

- 防作弊ACK机制下棋
- 局域网联机
- 支持创建房间
- 接入chatgpt
- 背景音乐
## 部署说明

需要将utils文件夹下的`constants.dart`中的`openaiApiKey`改为自己的。
